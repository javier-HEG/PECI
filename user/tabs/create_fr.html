Créez votre propre questionnaire

Grâce à cette boîte à outils, voujs pouvez créer un questionnaire pour l'évaluation de l'utilité. Vous pouvez ensuite par exemple mettre un lien vers ce questionnaire sur le site web de votre bibliothèque ou l'envoyer par e-mail ou par un newsletter. Les usagers de la bibliothèque remplissent alors le questionnaire et ils vous donnent ainsi un feedback par rapport aux services numérique que proposent votre bibliothèque.

Pour que vous ne deviez pas démarrer à zéro, vous trouverez sur cette plateforme un modèle très complet du questionnaire. Dans une première étape, vous êtes demandé à choisir à partir du modèle des question que vous voulez ajouter à votre propre questionnaire.

Dans une première étape, vous pouvez adapter les questions choisies au contexte de votre institution et ajouter d'autres questions. Une fois le questionnaire terminé, vous pouvez finir la cette phase. A partir de ce moment-là, il ne sera plus possible de remodifier le questionnaire. Vous obteznez alors un lien sous lequel le questionnaire est consultable. Vous pouvez intégrer ce lien sur un site web ou l'envoyer par e-mail.

Après que vous avez recolté un nombre suffisant de réponse et/ou le temps prédéfini pour le questionnaire à écoulé, vous pouvez terminer le sondage. Grâce à ça, plus personne ne pourrait participer au sondage.

La plateforme vous permet d'exporter les résultats dans un fichier .csv. Ce format est lisible par les logiciels de tableurs courants, tels que Microsoft Excel ou OpenOffice Calc.

Vous pouvez ensuite évaluer et analyser vos résultats.

Bonne chance!
