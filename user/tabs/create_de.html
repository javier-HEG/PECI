Kreieren Sie Ihren eigenen Fragebogen

Anhand dieses Toolkits können Sie Ihren eigenen Fragebogen für die Nützlichkeitsevaluation erstellen. Diesen Fragebogen können Sie danach beispielsweise auf der Website Ihrer Bibliothek verlinken oder per E-Mail oder Newsletter verschicken. Die Bibliotheksbenutzer füllen die Umfrage aus und geben Ihnen so ein Feedback zum digitalen Angebot Ihrer Bibliothek.

Damit Sie den Fragebogen nicht von Grund auf neu gestalten müssen, finden Sie auf dieser Plattform eine ausführliche Vorlage. Sie werden in einem ersten Schritt dazu aufgefordert, von dieser Vorlage diejenigen Fragen auszuwählen, die Sie in Ihren persönlichen Fragebogen übernehmen möchten.

In einem zweiten Schritt können Sie dann die ausgewählten Fragen an den Kontext Ihrer Institution anpassen und eigene Fragen hinzufügen. Ist der Fragebogen fertiggestellt, dann können Sie diese Phase beenden. Es ist dann nicht mehr möglich, weiterhin etwas am Fragebogen zu verändern.

Sie erhalten daraufhin einen Link, unter welchem ihre Umfrage abrufbar ist. Diesen Link können Sie auf einer Website integrieren oder per Mail verschicken.

Nachdem Sie genügend Antworten gesammelt haben und/oder die festgelegte Zeit für die Umfrage abgelaufen ist, können sie die Umfrage beenden. Dadurch kann danach niemand mehr an der Umfrage teilnehmen.

Die Plattform erlaubt es Ihnen, die Resultate in einer .csv-Datei zu exportieren. Dieses Format kann von gängigen Tabellenkalkulationssoftwares wie Microsoft Excel oder OpenOffice Calc lesbar.

Sie können nun Ihre Resultate auswerten und analysieren.
