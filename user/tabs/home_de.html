Willkomen auf der Plattform für Usefulness-Evaluationen von digitalen Angeboten im Bibliotheksbereich

Auf dieser Website finden Sie allgemeine Informationen zur Nützlichkeitsevaluation. Mit der hier vorgestellten Methode können Inhalte, Dienstleistungen und Funktionalitäten, welche von Bibliotheken online angeboten werden, evaluiert werden. Es geht dabei darum, folgende Fragen zu beantworten:

  • Werden die Inhalte, die den Benutzern zur Verfügung gestellt werden, von ihnen als nützlich empfunden?
  • Sollten die Inhalte anders dargestellt werden, um noch nützlicher zu sein?
  • Soll das Angebot weiter ausgebaut werden?
  • Soll eine Dienstleistung beibehalten werden?
  • Wird ein Bereich, obwohl er wenig benutzt wird, trotzdem als nützlich bezeichnet?
  • Soll die bisherigen Strategie für die Website beibehalten werden oder muss sie angepasst werden?

Die Plattform bietet Ihnen die Möglichkeit, auf einen vorgefertigten Fragebogen zurückzugreifen. Diesen können Sie Ihrem spezifischen Kontext anpassen und eigene Fragen hinzufügen. Darüber hinaus können Sie eine Umfrage direkt von dieser Plattform aus starten.

Die Plattform baut auf der OpenSource Software von LimeSurvey auf. Sie können deshalb den hier erstellen Fragebogen exportieren und ihn Sie können aber auch den hier erstellten Fragebogen exportieren und auf LimeSurvey bzw. LimeService importieren.

Die Resultate können Sie anschliessend als .csv oder als .xls exportieren.
